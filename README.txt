--------
neighpal
--------

Prototype betting odds manipulator


To run neighpal in dev environment:

 - localhost/stol.uk/neighpal/cgi-bin/neighpal.cgi

To release:

 - Upload neighpal to stol.uk/htdocs
 - Make sure that stol.uk/htdocs/cgi-bin/neighpal.cgi is executable (file permissions 755)
 - Remove from live site any occurrences of testpage.cgi and phpinfo.php

To run live:

 - http://stol.uk/neighpal
